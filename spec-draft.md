Goal:
An app that an ultimate teach coach can use to tracker players in a game and game time.

The sitch:
- The games are first to to 11, or end after 60 minutes(see rules below). 
- There are 5 players on the field at once. THere will be a number of Gx (girls/nonbinary) or bx (boys/nonbinary)
- Half time at 30 minutes past, or when the first team hits 6 points (whichever comes first) -- see below
- 10 minutes after a point starts, option to call timeout and swap players.

App capabilitis
- Need to be able to add players present at that game.
- Need to be able to select gender ratio for a point
- App should automatically determine who plays the next point based on algorithm to give fair play time to each player.
- Secondarily, vary the grade level of kids on the field (kids are grade 3,4, or 5)
- If we can find a way to track players over games, we could use +/- (net points scored for/against when a given player is on the field) as a way to even out each line-up (not putting all the best/worst players in one lineup/)
- For current point, we need to say which scored and move to the next point, thereby keeping score.
- App should know what the startime is and suggest when it's time to go to half or when end of game is coming up.
- App should know when it's been 10 minutes and suggest stopping to sub, if accepted, choose who the subs should be as if it were a point.
- Coach should be able to override algorithmic choices and see who on the roster has played how many points nad how long ago they played their last point.
- Should be able to preview points coming up and who is expected to play in those points.
- For each point, coach should be able to choose the gender ration (eg., 5bx/0gx, 4bx/1gx,3bx,2gx,etc)
- At start of game, choose a default ratio, or alternative ratio (e.g., 3bx/2gx followed by 2bx/3gx)

Optionally, during point coach can mark if a player had a great throw, D, assist (multiple can assist), or score. But this is a secondary thing.

It is VERY important that data doesn't somehow get lost midgame, eg., if browser crashes or coach hits back button or something. Need to save sessions and be able to roll back?
It would be great to sync with a spreadsheet (google sheet?) for storage.



League rules:
Point Total & Game Times:

Games are to 11 points. 
Mirror halftime at 6 points, and take a short timeout (3 minutes or less).
The hard cap is applied at 60 minutes . 
Teams should finish the current point (if between points, finish the next point).
The team that is ahead at the end of that point is the winner. 
If the game is tied at the end of that point, play one more point ("universe point").
At 70 mins, each team gains one more possession to try to score. If the first team scores, the game is done. If at the conclusion of this the game is still tied the game ends in a tie.
Half cap is at 30 minutes, if one team hasn't gotten to 6 by 30 minutes you use this rule. If one team has gotten to 6, you take the standard half. 
6.D.3. The halftime cap occurs once a predetermined time of play has elapsed. At the halftime cap, play continues until the current scoring attempt is completed. If, after the current scoring attempt is completed, the halftime target has not yet been reached by one team, one is added to the higher score and the resulting number is the new halftime target.
